# Sealos SRE Agent

Sealos SRE Agent with MCP protocol support for Kubernetes cluster management.

## Features

- Kubernetes cluster management
- MCP (Model Context Protocol) support
- Pod and namespace operations
- TypeScript implementation

## Installation

```bash
npm install
```

## Usage

```bash
# Build
npm run build

# Start server
npm run start:server

# Start client
npm run start:client
```
